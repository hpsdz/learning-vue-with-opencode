# Vue 学习计划与进度

## 一、Vue 学习计划

### 阶段 1: Vue 基础 (Essentials)
- **目标**: 掌握 Vue 的核心概念和基本用法，能够独立开发简单的单页应用。
- **内容**:
    - Vue 实例创建与选项
    - 数据绑定：插值表达式 `{{ }}`、`v-bind`
    - 指令系统：`v-if`, `v-else-if`, `v-else`, `v-show` (条件渲染)
    - 列表渲染：`v-for`
    - 事件处理：`v-on`
    - 表单输入绑定：`v-model`
    - 计算属性 (Computed Properties)
    - 侦听器 (Watchers)
    - 组件基础：组件注册、Props
    - 生命周期钩子

### 阶段 2: 深入组件 (Component In-depth)
- **目标**: 深入理解 Vue 组件化开发，掌握组件间通信的各种方式。
- **内容**:
    - 组件通信：Props (父子通信)、自定义事件 `emit` (子父通信)、`v-model` 在组件上的应用
    - 插槽 (Slots)：内容分发
    - Provide / Inject (跨层级通信)
    - 动态组件
    - 异步组件

### 阶段 3: 路由管理 (Vue Router)
- **目标**: 掌握 Vue.js 官方路由库的使用，实现单页应用的页面跳转和管理。
- **内容**:
    - Vue Router 安装与配置
    - 声明式导航 `<router-link>`
    - 动态路由匹配、嵌套路由
    - 编程式导航 `router.push()`
    - 路由守卫 (Navigation Guards)

### 4: 状态管理 (Vuex / Pinia)
- **目标**: 掌握大型应用中的状态管理方案，理解响应式数据流。
- **内容**:
    - Vuex 核心概念 (State, Getters, Mutations, Actions, Modules)
    - Vuex 基本使用
    - Pinia (Vue 3 推荐) 基础与使用

### 阶段 5: 工具链与生态 (Tooling & Ecosystem)
- **目标**: 熟悉 Vue 生态工具，能够搭建和优化项目开发环境。
- **内容**:
    - Vue CLI / Vite (项目构建工具) 的使用
    - 单文件组件 (Single File Components - SFC) 详解
    - 使用 `axios` 进行 HTTP 请求
    - 集成第三方 UI 库 (如 Element Plus, Ant Design Vue)

### 阶段 6: 高级主题与最佳实践 (Advanced & Best Practices)
- **目标**: 学习 Vue 的高级特性，了解项目优化和代码规范。
- **内容**:
    - 自定义指令
    - 混入 (Mixins)
    - 插件 (Plugins) 的开发与使用
    - 渲染函数 (Render Functions) 与 JSX
    - 性能优化策略
    - TypeScript 与 Vue (基础集成)

### 阶段 7: 项目实践 (Project Practice)
- **目标**: 通过实际项目巩固所学知识，提升解决实际问题的能力。
- **内容**:
    - 独立完成一个包含前端路由、状态管理、HTTP 请求的完整应用。
    - 阅读并理解优秀的 Vue 开源项目代码。
